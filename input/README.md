```
D T O V
-- D rows with depots coordinates --
x y
-- T rows with targets coordinates --
x y
-- O rows with obstacles coordinates --
x y
-- V rows with vehicles coordinates --
u v

D - number of depots
T - number of targets
O - number of obstacles
V - number of vehicles
x, y - geo positions coordinates, lat lng with dot floating separator
u - max distance, km
v - speed, km/h
```
Example:
```2 2 1 2
```
